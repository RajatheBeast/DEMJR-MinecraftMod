WHEN YOU TRY TO RUN THE MOD IN ECLIPSE OR INTELLIJ, YOU MAY GET A STUPID UUID STRING ERROR. 
SIMPLY IGNORE AND COMPILE THE MOD BY OPENING A COMMAND WINDOW IN THE MAIN FILE, TYPING "gradlew build". 
ONCE YOU HIT 82%, HIT CONTROL-C AND TERMINATE THE PROCESS.
GO INTO THE BUILD FILE, THEN INTO THE LIB FILE, THEN DRAG THE DEMJR JAR FILE OUT.
PLACE IT IN THE MODS FOLDER OF YOUR MINECRAFT WITH FORGE.
IF ALL THIS HAS BEEN DONE CORRECTLY, THE MOD SHOULD RUN WITHOUT A HITCH.
